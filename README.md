# blender-addon-iBake
Blender addon to automate high-poly to low-poly baking for VR assets

## What this tool does
Automates high-poly to low-poly texture baking for photogrammetry assets,
designed for VR optimization workflows.

## Problem
Manual baking and texture reassignment took ~30 minutes per asset
and was highly repetitive.

## Solution
This addon allows selecting a high-poly source mesh and a low-poly target
and performs baking and material assignment in one click.

## Tools
- Blender
- Python

## Use case
- VR projects
- Photogrammetry optimization
- Repetitive asset pipelines

## Note
This tool was created for production use.
Code written with assistance from AI and customized for project needs.
